# Day 2026-03-23 - Intro to CI with GitHub Actions

Objective

- Understand basics of GitHub Actions and how to set up a simple CI workflow.

Lesson

1. What is GitHub Actions: a brief overview.
2. Workflow files: YAML structure, triggers, jobs, steps.
3. Example: run tests on push using actions/checkout, setup-node, run npm test.
4. Secrets and tokens: where to store and how to use them (GITHUB_TOKEN vs PAT).

Exercise

- Add a .github/workflows/ci.yml to this repo that runs tests on push to main.
